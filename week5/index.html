Questions

  • What other testing modules are there besides Jest?

This list comes from a 2020 survey according to testim.io. Other than Jest, popular unit testing frameworks include: Mocha, Storybook, Cypress, and so on. They rate Mocha as complicated but powerful, able to unit test both back end and front end. Cypress is popular because it is browser-based and does not require set up or installation. Storybook focuses on documentation while testing components. In the end, the article labels Jest as the best unit testing framework since it is open source, quick to learn, and easy to use.

the ToDos prototype is loosely following Test Driven Development principles. Using a set of test data, the UI is built and tested. I intend to use this test data until the UI is functioning properly with marking and removing tasks from the ToDos.
