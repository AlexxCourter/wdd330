Week 7 Notes

principles practiced

  • How secure is AJAX?

Pure functions in functional programming follow 3 principles. 1, return value of the function is dependent on values provided as arguments (no globals in the function body). 2, the function doesn't change any values or data in other parts of the program. It only makes data transformations internally and returns new values rather than altering outer data from the program. 3, given the same arguments, the function will return the same result. Because of these principles, pure functions must have at least one argument and a return value.
