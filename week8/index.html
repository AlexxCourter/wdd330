Questions

  • How secure is AJAX?

To answer the question, "How secure is AJAX development?", I found some information on tutorialspoint.com that stated "AJAX-based Web applications are subject to the same security threats as regular web applications". From this, I infer that security measures must be taken similiar to any web application, providing countermeasures against bad actors who either search for vulnerabilities in the program or inject malicious code such as via a form (SQL Spoofing can crack weak log in methods).
